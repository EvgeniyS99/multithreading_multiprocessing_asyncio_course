# Lesson 1: OS and CPU Recap

This lesson reviews CPU and memory organization, user and kernel space,
processes, threads, scheduling, process states, and the distinction between
I/O-bound and CPU-bound work.

## Materials

- [Presentation](01_os_cpu_recap.pdf)
- [OS and process context notebook](notebooks/os_cpu_recap.ipynb)
- [`process_context.py`](code/process_context.py) — inspect the current process,
  thread, CPU count, and CPU affinity where supported

Run the code example with:

```bash
python code/process_context.py
```
